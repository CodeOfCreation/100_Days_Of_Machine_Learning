# CampusX: 100 Days of Machine Learning Archive

## Overview
This repository serves as the definitive record of all instructional materials, practical exercises, and project implementations completed during the **CampusX '100 Days of Machine Learning'** program. It systematically documents the acquisition of competencies in data science, machine learning engineering, and artificial intelligence, adhering to a rigorous daily learning schedule.

## Curriculum Scope
The contents are organized chronologically and thematically to reflect the program’s instructional sequence:

- **Days 1–20: Foundations & Data Manipulation**
  - Python programming essentials for data science.
  - Advanced proficiency in NumPy, Pandas, and Matplotlib/Seaborn.
  - Exploratory Data Analysis (EDA) techniques and data cleaning pipelines.

- **Days 21–50: Classical Machine Learning**
  - Supervised Learning: Linear/Logistic Regression, Decision Trees, SVM, KNN, Naive Bayes.
  - Unsupervised Learning: K-Means Clustering, Hierarchical Clustering, PCA.
  - Model Evaluation: Cross-validation, hyperparameter tuning (GridSearch/RandomSearch), and bias-variance tradeoff analysis.

- **Days 51–80: Advanced Algorithms & Ensemble Methods**
  - Boosting algorithms: AdaBoost, Gradient Boosting, XGBoost, LightGBM, CatBoost.
  - Feature engineering strategies and selection techniques.
  - Handling imbalanced datasets and advanced preprocessing pipelines.

- **Days 81–100: Deep Learning & Deployment**
  - Neural Network fundamentals with TensorFlow/Keras.
  - Convolutional Neural Networks (CNNs) for computer vision tasks.
  - Natural Language Processing (NLP) basics and sequence models.
  - Model deployment using Streamlit, Flask, or FastAPI.

## Repository Structure
- `01_Python_Foundations/`: Core programming concepts and problem-solving scripts.
- `02_Data_Analysis/`: Notebooks demonstrating EDA, visualization, and data wrangling.
- `03_Machine_Learning/`: Implementations of classical algorithms from scratch and via scikit-learn.
- `04_Deep_Learning/`: TensorFlow/Keras projects involving image classification and NLP.
- `05_Projects/`: End-to-end capstone projects integrating full ML pipelines.
- `Resources/`: Curated lecture notes, cheat sheets, and reference documentation.

## Technical Stack
- **Languages:** Python 3.x
- **Libraries:** NumPy, Pandas, Scikit-learn, Matplotlib, Seaborn, TensorFlow, Keras, Streamlit
- **Tools:** Jupyter Notebook, VS Code, Git/GitHub
